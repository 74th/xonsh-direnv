`direnv <https://direnv.net>`_ support for the `xonsh shell <https://xon.sh>`_

https://direnv.net

https://xon.sh

This works with python 3.7.0, xonsh 0.8.8 and direnv 2.17.0.

::

	xpip install xonsh-direnv
	xontrib load direnv
